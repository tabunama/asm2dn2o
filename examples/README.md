# Examples

This folder contains public example scripts for using `asm2dn2o`.

## Available examples

- `quickstart_aeration_tank.py`  
  Minimal reactor example.

- `one_day_example_from_project_data.py`  
  One-day replay using project `.npz` inputs.

- `full_dataset_example_from_project_data.py`  
  Full-dataset replay using project `.npz` inputs.
