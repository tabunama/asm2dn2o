# asm2dn2o

Python package for ASM2d-N₂O dynamic process modelling.

## Installation

This package is currently distributed as a compiled wheel for:

- Windows x64
- CPython 3.12

Install from PyPI:

```bash
pip install asm2dn2o

Import from Python:

.. code-block:: python

   from asm2dn2o import asm2d_n2o, clarifiers, combiner, delay
