Single Tank
====================

Purpose
-------

Single Tank example.

Objective
---------

Build one ASM2d-N2O reactor with:

- one 56-state inlet stream,
- a constant ``KLa``,
- a pH value or pH time series,
- one explicit time loop.



Recommended outputs
-------------------

Plot at least:

- ``S_O2``
- ``S_NH4``
- ``S_NO3``
- ``S_N2O``
