bas_par_sets.py
===============

Purpose
-------

``bas_par_sets.py`` stores the long kinetic and stoichiometric parameter vectors for
the reactor kernels. The current helper returns separate AR and AX vectors from
combined matrices for ``ss``, ``dy``, and ``120d`` modes. The current arrays contain
182 entries per basin. fileciteturn16file4

How the helper is used
----------------------

.. code-block:: python

   PAR_AR, PAR_AX = build_PAR_sets(mode)

The combined matrices are ordered column-wise as ``[AR, AX]``. This structure is
helpful because it keeps both basins aligned against the same parameter index map.

Important documentation rule
----------------------------

Not every parameter should be treated as a calibration knob:

- some entries are stoichiometric constants,
- some are elemental content coefficients,
- some are transport constants,
- some are clearly calibration-sensitive operational knobs.

When tuning the model, document **which subset** you changed and keep the full vector
order intact.
